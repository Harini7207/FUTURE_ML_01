# Sales Forecasting using Machine Learning

## Project Overview
This project predicts future sales using historical Superstore data.  
A machine learning model is trained to analyze past sales trends and forecast future sales for the next months.

## Dataset
The dataset used is the Superstore dataset containing information such as:
- Order Date
- Sales
- Category
- Region
- Profit

The dataset is processed to create time-based features like Month and Time Index for forecasting.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Steps
1. Data Loading
2. Data Cleaning and Preprocessing
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Sales Forecasting for the next 3 months
7. Visualization of Actual vs Predicted Sales

## Model Used
Linear Regression model was used to predict sales based on time index and month features.

## Model Performance
Mean Absolute Error (MAE): 11599.75

## Results
The model successfully predicts sales trends and generates forecasts for future months.
